# README

A current version of the server is deployed at https://mk28.github.io/TINF21C_Team2_AAS_digital_nameplate/. 

## REQUIREMENTS
- Web-Server i.e.
  - [serve](https://www.npmjs.com/package/serve) for Node.js
  - [express](https://www.npmjs.com/package/express) for Node.js

## Starting the Application with serve

1. Install Node.js (at leas LTS version)
2. Open a terminal
3. Execute `npm i -g serve`
4. Navigate into the EXECUTABLE folder in the terminal
5. Execute `serve -s`
6. Open a browser at the address shown in the terminal
